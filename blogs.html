Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM using getElementById, getElementsByClassName, querySelector and querySelectorAll


Question-2: What is the difference between innerHTML, innerText, and textContent?

By using innerHTML we can dynamically add html elements and by using innerText we can change or add the text of the html element and by using textContent we can change the text content of the element.


Question-3: What is event delegation in the DOM?

In event delegation we add event listener to the parent instead of the children so that we can easily change the properties of the children by getting the parent element.


Question-4: What is event bubbling in the DOM?

In event bubbling we can go up from a child to its parent.


Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement() method, we can add elemsts using document.appendChild() method and we can remove elements using document.removeChild() method.
